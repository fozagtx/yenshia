# Yenshia Stellar Contracts

`yenshia_ultrahonk_verifier` is the Soroban verifier for Yenshia's Noir
UltraHonk private location proof.

The contract stores one verification key at deployment and exposes
`verify_location_proof(public_inputs, proof_bytes)`. Proofs and verification keys are
generated by the Noir/Barretenberg circuit under `circuits/yenshia_location`.
